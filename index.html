IBM Bluemix Developer Advocates

What's Bluemix? IBM's Bluemix cloud products are a set of cloud runtime platforms which enable you to deploy applications on the web quickly and easily. Bluemix also hosts a number of services which can be used by those applications.

To find out more about Bluemix, visit http://bluemix.net

Bluemix Developer Advocates help customers become successful with their use of IBM's Bluemix cloud products.

what

The Github organization "IBM-Bluemix" contains a number of projects authored by IBM's Bluemix Developer Advocates.

bluemix-hello-iojs-container

sample iojs server that can be run on Bluemix Containers

bluemix-cloud-connectors

Spring connectors for some Bluemix services

bluemix-python-flask-sample

A sample application that uses the python flask microframework and can be deployed to IBM Bluemix.

cf-models

access Cloud Foundry models from node

iot-sensor-tag

Sample code for connecting to a TI Sensor Tag and using that data in apps deployed to Bluemix.

news-aggregator

Bluemix.info - News Aggregator for Bluemix Developers

iot-raspberrypi

This repository contains sample(s) for connecting Raspberry Pi devices to the IBM Internet of Things Foundation

todo-apps

Sample ToDo apps writtin in various languages

zendcon-2014-hackathon

Sample PHP code for IBM Bluemix based applications with services including NoSQL, email, and more.

cf-manifest-generator

source for the http://cfmanigen.mybluemix.net/ site.

bluechatter

An IRC-like / chat application

auto-scaling-demo

A demo for integrating auto-scaling into your Cloud Foundry application
