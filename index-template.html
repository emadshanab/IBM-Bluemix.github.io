IBM Bluemix Developer Advocates

What's Bluemix? IBM's Bluemix cloud products are a set of cloud runtime platforms which enable you to deploy applications on the web quickly and easily. Bluemix also hosts a number of services which can be used by those applications.

To find out more about Bluemix, visit http://bluemix.net

Bluemix Developer Advocates help customers become successful with their use of IBM's Bluemix cloud products.
